• 🎨 Modern UI Design  
• 📱 Responsive Layout  
• 🔄 Easy timer switching  

---

# 🧠 What is Pomodoro Technique?

The **Pomodoro Technique** is a productivity method that uses timed intervals to improve focus and efficiency.

Typical cycle:

25 minutes → Focus  
5 minutes → Short Break  
After 4 cycles → Long Break

This helps reduce burnout and increase concentration.
